Sobre Nosotros

Delegando el maquetado completo a la IA

Somos un grupo de estudiantes de la Tecnicatura Superior en Desarrollo de Software que decidió explorar los límites de las herramientas de desarrollo impulsadas por Inteligencia Artificial. Nuestro desafío es claro: construir interfaces completas delegando cada decisión de maquetación, estilo e interactividad al agente, utilizando únicamente Prompt Engineering.

No abrimos el editor para corregir CSS ni ajustar JavaScript. Cada sección, animación y detalle visual que ves en esta página es el resultado directo de instrucciones precisas enviadas al agente. Medimos qué tan autónomo es realmente un asistente de código cuando se le confía un proyecto de principio a fin.

Este experimento nos permite comparar la calidad del output generado, la coherencia del diseño y la capacidad del agente para interpretar restricciones técnicas complejas sin intervención humana en el código fuente.

Metodología Zero-Touch

Prompt detallado, output directo. Sin refactorización manual ni parches post-generación.

  • Especificación de diseño en lenguaje natural
  • Validación visual del resultado
  • Documentación de autonomía alcanzada
Características Evaluadas

Tres pilares del desafío

Cada agente es evaluado bajo criterios objetivos que miden su capacidad real para actuar como desarrollador frontend autónomo.

Autonomía de Código

Medimos si el agente puede generar HTML semántico, CSS responsive y JavaScript funcional sin solicitar aclaraciones adicionales ni depender de correcciones iterativas del usuario.

92% autonomía

Precisión del Prompt

Evaluamos qué tan bien el agente interpreta restricciones explícitas: paleta de colores exacta, tipografías específicas, prohibición de frameworks y estructura de secciones obligatorias.

88% precisión

Manejo de Tokens de Contexto

Analizamos la eficiencia del agente para mantener coherencia visual y estructural a lo largo de archivos extensos, optimizando el uso del contexto disponible.

85% eficiencia
