Sobre Nosotros

Un laboratorio estudiantil para medir criterio, contexto y ejecución.

Somos un grupo de estudiantes que está poniendo a prueba herramientas de desarrollo impulsadas por IA. El desafío consiste en delegar el maquetado completo de una landing page mediante ingeniería de instrucciones, sin intervenir manualmente el código producido por el agente.

Observamos cómo interpreta restricciones visuales, organiza archivos, responde a cambios de contexto y mantiene coherencia entre HTML, CSS y JavaScript. Cada prompt funciona como una especificación técnica y cada resultado se analiza como una experiencia real de colaboración humano-IA.

Características Evaluadas

Tres ejes para entender la autonomía de un agente.

Autonomía de Código

Capacidad del agente para estructurar archivos, resolver layout, aplicar estilos y conectar interacciones sin asistencia manual.

Precisión del Prompt

Nivel de fidelidad con el que la IA interpreta requisitos, jerarquías de contenido, tono visual y restricciones técnicas.

Manejo de Tokens de Contexto

Eficiencia para conservar instrucciones clave, priorizar datos y sostener consistencia durante todo el flujo de generación.
